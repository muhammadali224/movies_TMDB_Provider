
# Lumovies

A Flutter project Integrate TMDB API to View movies ,tv show and more .


## Features

- Live previews
- Fullscreen mode
- Cross platform
- Rating Video Or TV Show
- View User Rating
- Add Movies Or TV To Favorite
- Search for Movies and TV
